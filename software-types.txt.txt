software types
System Software – Runs and manages the computer (e.g., operating systems like Windows, macOS).

Application Software – Programs for user tasks (e.g., Word, games, browsers).

Utility Software – Tools that help maintain the system (e.g., antivirus, backup tools).

Web Applications
Web applications run through a web browser and don’t need to be installed on the device.

They can be accessed from anywhere as long as there is an internet connection.

Examples include email platforms, online shopping sites, and cloud-based tools like Google Docs.

Mobile applications are software designed to run on smartphones and tablets.

They provide services or features like messaging, navigation, shopping, or entertainment.

They are downloaded and installed from app stores such as Google Play or the Apple App Store.
Desktop applications are software programs installed directly on a computer or laptop.

They often offer powerful features and can work without an internet connection.

Examples include Microsoft Word, Photoshop, and video editing programs.
